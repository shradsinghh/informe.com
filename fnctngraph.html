FUNCTIONS

  1. VOCABULARY

    1. Range: correspondance between two sets(sets of order pairs)
    2. Function: a relation in which every input(x) has exactly one output(y)
    3. Domain: set of all possible values for x
    4. Range: set of all images of elements in the domain(all values for y)
    5. Input: x-value
    6. Output: y-value
    7. Independent: (argument)variable x(values from domain)
    8. Depenedent: Variable y (value depends on x)
    9. Expression: mathmatical phrase that contains numbers, variables and operators
    10. Equation: mathematical sentence states that two expressions are equal.
